# 01 · Éligibilité — MBOLO Santé vs critères TEF

## Critères d'éligibilité officiels (cycle 2026)

| Critère TEF | Exigence | MBOLO Santé | Statut |
|-------------|----------|-------------|--------|
| Nationalité / résidence | Citoyen **ou résident légal** d'un pays africain | Gabon 🇬🇦 | ✅ |
| Âge | **18 ans minimum** | À confirmer (toi / fondateur) | ✅ (si ≥ 18) |
| Stade de l'entreprise | **0 à 5 ans** d'existence | Early-stage / lancement | ✅ |
| But lucratif | Entreprise **commerciale** (pas ONG) | Marketplace à commissions | ✅ |
| Localisation d'activité | Opère **principalement en Afrique** | Libreville, Gabon | ✅ |
| Secteur | **Tous secteurs** acceptés | HealthTech / logistique de soins | ✅ (secteur à fort impact) |
| Candidature individuelle | 1 candidature par personne / entreprise | — | ✅ |

> **Conclusion : MBOLO Santé est pleinement éligible.**

## Points de vigilance avant de postuler

1. **Enregistrement de l'entreprise** — Le capital d'amorçage (5 000 USD) n'est versé
   qu'à une entreprise **formellement enregistrée**. Si MBOLO n'est pas encore
   immatriculée au Gabon (RCCM / NIF), lance la démarche **avant** la sélection.
   *(L'enregistrement n'est pas exigé pour postuler, mais l'est pour recevoir les fonds.)*

2. **Compte bancaire / mobile money professionnel** au nom de l'entreprise (pour
   recevoir le décaissement).

3. **Un seul fondateur candidate** au nom du projet (le programme finance la personne
   entrepreneure, pas « la société » abstraitement). Choisis le porteur principal.

4. **Langue** — La candidature et la formation existent **en français** : sélectionne
   le français sur TEFConnect.

## Ce qui rend MBOLO particulièrement crédible aux yeux du jury

- **Produit déjà construit** (app mobile multi-rôles + API + dashboard admin) → preuve
  d'exécution, pas seulement une idée. **C'est un différenciateur majeur.**
- **Conformité réglementaire intégrée** : la plateforme n'est jamais vendeur du
  médicament ; le pharmacien valide toute ordonnance. → réduit le risque juridique.
- **Création d'emplois** directe : coursiers, et partenaires (pharmacies, cuisines,
  médecins) → aligné avec la mission TEF (emplois + impact).
